# 思路
## 抽象为数组
* 初始是‘0000’，设为初始结点
* 密码4位，每次转动4个位置都有可能转动，同时每位可以向上⬆️或者向下⬇️转动，所以第二层有4x2=8个结点
* 第二层的8个结点继续向下尝试，以此类推
## 注意
每尝试一个组合就要将该组合放置进“死亡数字”deadends中，避免重复搜索
